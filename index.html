Spring Boot 3 · Java 17 · 后端实习路线

从自动装配到权限控制的后端工程训练

这是一套面向实习课程的 Spring Boot 后端学习项目。课程从自动配置与自定义 Starter 开始， 进入 MyBatis-Plus 数据持久化，再完成 Sa-Token 登录认证与接口鉴权。

查看知识地图 开始学习
git clone https://gitee.com/flower-dance-butterfly/experi2026.git cd day01 mvn spring-boot:run # Day02: 数据查询 # Day03: Token 认证 + 权限校验

点击节点可跳转到对应课程章节。

课程章节

每一节课都对应一个独立工程，可单独运行。

Day 01 · 自动装配与自定义 Starter

理解 Spring Boot 为什么能做到开箱即用，并完成一个可配置的日志工具 Starter。

核心知识点
  • IoC 与依赖注入
  • 条件装配注解
  • ConfigurationProperties
  • AutoConfiguration.imports
cd day01 mvn spring-boot:run

Day 02 · MyBatis-Plus 数据持久化

围绕用户查询、分页查询和条件查询，掌握后端项目中的数据访问层写法。

核心知识点
  • Entity / DTO / Mapper / Service / Controller
  • BaseMapper 与 IService
  • QueryWrapper 条件查询
  • Page 分页插件
cd day02 mvn spring-boot:run

Day 03 · Sa-Token 认证与鉴权

实现登录认证、Token 校验、角色权限控制和全局异常处理。

核心知识点
  • StpUtil.login()
  • SaInterceptor 路由拦截
  • @SaCheckPermission
  • @SaCheckRole
cd day03 mvn spring-boot:run

运行指南

按课程顺序启动，也可以单独运行某一天的工程。

环境准备
安装 JDK 17、Maven 3.6+。Day 02 需要 MySQL 8.0，并提前创建数据库。
启动项目
cd day01 mvn spring-boot:run
接口测试
使用仓库中的 Postman 集合测试接口。
POST http://localhost:8081/user/doLogin?username=admin&password=123456
GET http://localhost:8888/yyzx/user/page?pageNum=1&pageSize=10
